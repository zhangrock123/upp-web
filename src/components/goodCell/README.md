# goodCell

## 商品详情卡片

### props

|参数|类型|是否必须|说明|备注|
|---|---|---|---|---|
|isLink|Boolean|否|点击跳转商品详情|默认false|
|isGroup|Boolean|否|是否是参团|默认false，影响价格显示,true=>团购价格,false=>原价|
|goodData|Object|否|商品信息|默认{}|
|isOrder|Boolean|否|是否是参团|默认false，影响数据格式化方法|
|noDisplay|Object|否|过滤显示信息|默认{}，全显示|
